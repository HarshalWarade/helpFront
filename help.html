NOW

The first button given on the website other than on the navbar is Get Snippet. Just copy the whole snippet and paste it into your empty HTML document. It will enable your document to use all the features of this website.

The further options provided when you scroll through the site are navbars, buttons, cards, and more will be added soon.


Navbars : This are the ones that appears on the extreme top (not necessary) of your website. Just copy and best the code of any navbar from this website to your document. Some navbars needs the addition of font awesome links to your page, they're provided above.


Buttons : The second option is buttons, here you can select any of the given buttons on the site, the first category of buttons is simple one with background. The second category has an ouline to them rather than background.


The intresting part: CARDS

Now the major part that I hope you'll be using is grids, or flex. We have provided huge variety of options for you to play around with your code. How can you do that?

Well, there is a class in group of cards named as grouped_h_cards_rpt3. Now, this is originally grouped_h_cards_rpt. The number 3 there means, arrange the child cards in columns of 3. You can change the 3 to 4, this would mean that arrange the cards in four columns. The maximum number of columns provided are 6



Understanding the padding classes:

We have provided pre-defined padding classes which basically, in their basic form are h_px, in this x and y components are added, seperated by a hyphen (-). For example: h_px-4-0, this means add padding as 40px 0px or in a simple form, it will add padding as 40px 0 40px 0. Another example: h_px-1-2, this means include padding as 10px 20px, which basically means padding as 10px 20px 10px 20px. You can add any number here, not exceeding than 60px in each side.



Understanding the height classes:

The site has pre-defined height classes for you. The base form of height class is h_hgt, the height is added by adding a hypen in between. For example: h_hgt-2, this means give the element a height of 20vh, similarly h_hgt-6, means give the element a height of 60vh. The maximum limit for the height class is 10, which means 100vh.
